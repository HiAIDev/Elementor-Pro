# Menu

> 封装默认 Menu 通过传入路由或者自动获取路由生成导航，与 `vue-router` 高度绑定

## 使用

```vue
<template>
  <pro-menu />
</template>
```

## 配置

| 参数   | 说明                                                                | 类型  | 默认值                 |
| :----- | :------------------------------------------------------------------ | :---- | :--------------------- |
| routes | 自定义生成侧边菜单栏的路由，同 `vue-router` 中 routes (仅扩展 meta) | array | 从 `vue-router` 中获取 |

## 插槽

| name | 说明                                                 |
| :--- | :--------------------------------------------------- |
| -    | 控制菜单显示内容，参数为 { meta, path, redirect } 等 |
